Welcome to HAI GUI Framework's documentation!
=============================================

**HaiGF** (High energy physics Artificial Intelligence Graphical Framework) is a python package for building graphical user interfaces 
based on HAI algorithm library for high energy physics analysis. It is based on the `PyQt5` library and the `Qt5` framework.

Feautures
---------

+ Modular design, easy to extend, easy to use.
+ The front and end are completely separated based on the `HAI` library so that the application will be lightweight.
