Main UI

Main Window

This is the main window of the hai gui framework, inherited from QMainWindow. Alias: mw.

Usage:

import sys
from PySide2.QtWidgets import QApplication
from HaiGF import HMainWindow

app = QApplication(sys.argv)
mw = HMainWindow()
mw.show()
mw.raise_()
sys.exit(app.exec_())
mw.install_plugin(plugin: HPlugin)

Call the install method of the custom plugin to install it.

Parameters

plugin – A plugin instance in inherited from HPlugin.

Core Func Bar

This is the Core Function Bar of the main window, inherited from QToolBar.

Alias: cfb, mw.cfb.

  • index – int

Main Side Bar

This is the main side bar of the main window, inherited from QDockWidget.

It contains a title bar with actions and a widget.

Alias: msb, mw.msb.

msb.add_widget(widget: HMainSideBarWidget, action: QAction)

Add a widget corresponding to an action (in the core func bar) to the main side bar.

The widget will be loaded when the action is triggered.

Central Widget

This is the central widget of the main window, inherited from QWidget. It contains a grid layout, which contains multiple splitter. The splitter split tab widgets, which contains multiple tab and pages.

Inherit relationship (parent to child)：CentralWidget -> Splitter -> QTabWidget -> Tab and Page

Alias：cw, mw.cw

cw.add_tab_widget(tab_widget: HTabWidget, *args)

Add one tab widget to central widget and asign it to a splitter autumaticly.

Parameters

tab_widget – HTabWidget

Example:

from HaiGF import HTabWidget
from HaiGF import HPage

tab_widget = HTabWidget(parent=cw)  # create a tab widget
page = HPage(parent=cw,
            title='page_title',
            icon='icon_name')  # create a page
tab_widget.add_page(page)  # add a page into tab widget
cw.add_tab_widget(tab_widget)  # add a tab widget into central widget
cw.add_page(page: HPage, tabw_idx: Optional[int] = None, *args)

Add one page to tab widget.

Parameters

  • page – HPage

  • tabw_idx – int, specify which tab widget to add page, default is None, means add to current tab widget.

Example:

from HaiGF import HPage

page = HPage(parent=cw,
            title='page_title',
            icon='icon_name')  # create a page
cw.add_page(page)  # add a page into current tab widget on central widget
