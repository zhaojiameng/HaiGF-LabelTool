HGF

HaiGF.HGF are the colors, fonts, css, and themes presets.

Usage

from HaiGF import HGF
HGF.COLORS.White  # lots of colors presets
HGF.FONT_FAMILY
HGF.FIRST_LEVEL_FONT
HGF.FIRST_LEVEL_TLTILE_CSS
HGF.COLORS()

Those are the colors presets with #RRGGBB format.

Colors:

White, WhiteSmoke, Gainsboro, LightGray, Silver, DarkGray, Gray, DimGray, LightSlateGray, SlateGray, DarkSlateGray, LightBlack, Black Blue, MediumBlue, MidnightBlue, DarkBlue, Navy, RoyalBlue, CornflowerBlue, LightSteelBlue, LightSlateBlue, SlateBlue, DarkSlateBlue, Indigo, DarkOliveGreen

Usage:

HGF.COLORS.White
