HPage

HaiGF的基本页面类，继承于QWidget，可添加到中央控件中。

一个页面包含一个标题栏和一个空的控件。

标题栏包含标题图标、标题文本、关闭按钮等。

Example:

from HaiGF import HPage

class CustomerPage(HPage):
    def __init__(self, parent=None, **kwargs):
        super().__init__(parent, **kwargs)
        self.set_title(self.tr('Customer Page'))
        self.set_icon(<QIcon>)

        self.setup_ui()

    def setup_ui(self):
        # setup customer ui here (QWidget).
        pass
HPage.set_icon(icon: QIcon)

设置页面标题的图标

HPage.set_title(title: str)
