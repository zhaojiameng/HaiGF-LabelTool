HaiGF - Hep AI Gui Framework

HaiGF (High energy physics Artificial Intelligence Graphical user interface Framework) is a framework for building graphical user interfaces based on HAI algorithm library for high energy physics analysis. It is based on the PyQt5 library and the Qt5 framework.

Interface

Feautures

  • Modular design, easy to extend, easy to use.

  • The GUI front and AI algorithm backend are completely separated based on the HAI library so that the application will be very lightweight.
